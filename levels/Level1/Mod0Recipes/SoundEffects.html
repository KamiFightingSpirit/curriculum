Sound Effects Machine


Goal: Make a GUI program using Swing in a Model-View-Controller design architecture.

The program will have a window (frame) with a number of buttons that will play different sound effects when clicked.


  1. Create a GUI for a sound effects machine that will play the different sounds (see above) when buttons are pressed. You will need FOUR classes to complete this program as follows:

    • a Model

      that will have the sound effects (see the Drum Kit recipe for setting up sound (AudioClip) objects).

      Gather sound bites for your sound effects machine. They must be .wav files. Free sound effects can be found here. The League also has an account at freesound. Ask your teacher for our login.

      The .wav files must be placed into your sound effects recipe package (drag and drop).

    • a View

      that will have a frame, a panel, and buttons. NOTE: because this GUI has more than one button, you must add a JPanel to your JFrame and put the buttons on the panel! The View will need to have a variable for the controller (because it will need a mouse listener).

    • a Controller

      that is a mouse listener so it can process button presses. The controller acts as the go-between so needs to have variables for the view and model

    • a Runner

      that will contain the main method and create the View object at the start of the program
    If you need help to get started, take a look at the previous recipes that you already completed in this module. Drum Kit is a good one to look at since it also plays sound files and will have a lot of methods you can use.
