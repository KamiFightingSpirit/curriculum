Sound Effects Machine


Goal: Make a program that uses Model-View-Controller (MVC) design architecture.

This program will have a window (frame) with a number of buttons that will play different sound effects when clicked.


    If you need help with any of these steps, take a look at the previous recipes in this module.

    Complete the Model


  1. A sample sound effect (drum) has been provided. Add some more of your own here.

    They must be .wav files. Free sound effects can be found here. The League also has an account at freesound. Ask your teacher for our login.

    The .wav files must be placed into your sound_effects_machine package (drag and drop).

    2. Make the View


  2. Find the SoundEffects class in the sound_effects_machine package. Complete the makeFrame() method to create a frame, a panel, and a button for each sound effect in your machine. NOTE: you must add a JPanel to your JFrame because there will be more than one button! Make a variable for the controller (which will be a MouseListener).

    3. Runner


  3. Run the SoundEffectsRunner class (has a main method). Make sure it is showing your frame with buttons before proceeding.

    4. Connect a Controller


  4. Find the SoundEffectsController class. Make it a MouseListener so it can process mouse clicks.
  5. Create and initialize variables for the view and model objects where shown.
  6. If you successsfully made it a MouseListener, there should be a mouseClicked method. Add code to this method for each of the buttons on the view. Hint: use the button text to figure out which one was clicked
  7. Call the method on the Model that will return the sound (AudioClip) you want, then tell the AudioClip to play.
  8. Run the SoundEffectsRunner and check that each of the buttons plays the right sound.
