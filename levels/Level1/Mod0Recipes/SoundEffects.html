Sound Effects Machine


Goal: Make a program that uses Model-View-Controller (MVC) design architecture.

This program will have a window (frame) with a number of buttons that will play different sound effects when clicked.


    If you need help with any of these steps, take a look at the previous recipes in this module.
    All the classes for this program are in the in the sound_effects_machine package.

    Complete the Model


  1. Find the SoundEffectsModel class. A sample sound effect (drum) has been provided. Add some more of your own here.

    They must be .wav files. Free sound effects can be found here. The League also has an account at freesound. Ask your teacher for our login.

    The .wav files must be placed into your sound_effects_machine package (drag and drop).

    2. Make the View


  2. Find the SoundEffects class. Complete the makeFrame() method to create a frame, a panel, and a button for each sound effect in your machine. NOTE: you must add a JPanel to your JFrame because there will be more than one button! Make a variable for the controller (which will be a MouseListener).

    3. Runner


  3. Run the SoundEffectsRunner class (has a main method). Make sure it is showing your frame with buttons before proceeding.

    4. Connect a Controller


  4. Find the SoundEffectsController class and follow the instructions to complete it. Don't forget to add this MouseListener to all the buttons on the View!
  5. Run the SoundEffectsRunner and check that each of the buttons plays the right sound.
