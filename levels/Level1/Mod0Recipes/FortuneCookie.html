Fortune Cookie


Goal: Make a program from scratch that has a View (JFrame) and Controller (ActionListener).

The View will present a window (frame) with a button that will cause one of five random fortunes to pop up when the mouse is clicked on it.


  1. Make a FortuneCookie class in the fortune_cookie package, with a constructor and a makeFrame() method like this: 
       public class FortuneCookie {
    
        public FortuneCookie() {
            makeFrame();
        }
        
        public void makeFrame() {
            System.out.println("Making Frame");
       }
    }
  2. Now make a FortuneCookieRunner class to run this program. Runner classes have a main method. Look for the "Making Frame" message on the console when you run this. 
        public class FortuneCookieRunner {
        public static void main(String[] args) {
            new FortuneCookie();
        }
    }

    Make the View (Frame with a Button)

  3. Go back to the FortuneCookie class and put this code into the makeFrame() method in place of the print statement. This code creates a JFrame and gets it to show up when you run the program, but it will be very small until you add Components to it. 
        JFrame frame = new JFrame();
    frame.setVisible(true);
    frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
  4. Add this code to create a JButton and add it to the frame. 
        JButton button = new JButton("This is a button");
    frame.add(button);
    frame.pack();
  5. Run the program again. Does the button show up?

    6. Connect a Controller (ActionListener)


  6. Make a FortuneCookieController class that implements ActionListener. Eclipse can help you to import ActionListener into this program. Next, Eclipse can help you to add the "unimplemented method" for an ActionListener. Check that you now have a method called actionPerformed() in this class.
  7. Inside the actionPerformed() method, put code that will show a JOptionPane pop-up that says "Woohoo".
  8. Go back to the FortuneCookie class. Declare and initialize a member variable for a new Controller object, like this: 
        FortuneCookieController controller = new FortuneCookieController();
  9. To connect the controller to the button, add it as an ActionListener to the button like this: 
        button.addActionListener(controller);
  10. Run the program again. Click on the button. What happens? You should get the Woohoo pop-up.

    11. Now make the pop-up deliver a random message


  11. Go back to the FortuneCookieController class. In the actionPerformed method, generate a random number between 0 and 4: 
       int rand = new Random().nextInt(5);
  12. Depending on the value of rand, show one of the five different fortunes in a pop-up.
  13. [optional] Have other students run your program to receive their fortunes.
