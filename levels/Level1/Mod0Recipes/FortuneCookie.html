Fortune Cookie


Goal: Make a Swing program from scratch.

The program will have a window (frame) with a button that gives one of five random fortunes when clicked.


  1. Make a FortuneCookieView class in the fortune_cookie package, with a constructor and a makeFrame() method like the one below: 
       public class FortuneCookieView {
    
        public FortuneCookieView() {
            makeFrame();
        }
        
        public void makeFrame() {
            System.out.println("Making Frame");
       }
    }
  2. Now make a runner class to run the code above, like the one below and run the progrem. Check the console for the message. 
        public class FortuneCookieRunner {
        public static void main(String[] args) {
            new FortuneCookieView();
        }
    }
  3. Now change the makeFrame() method in FortuneCookieView, to make a JFrame and get it to show up when you run the program and set the default close operation to exit. Like this: 
        frame.setVisible(true);
    frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
  4. Make a JButton and add it to the JFrame. 
        JButton button = new JButton("This is a button");
    frame.add(button);
    frame.pack();
  5. Run the program again. Does the button show up?

    6. The next steps will make the view react when the user clicks the button by adding a controller


  6. Make a FortuneCookieController class that implements ActionListener. Eclipse can help you to import the ActionListener and add the "unimplemented methods". Check you now have an actionPerformed method in this class.
  7. Put a JOptionPane pop-up inside the actionPerformed() method that says "Woohoo".
  8. Add a member variable to the FortuneCookieView class to hold the controller object. 
        FortuneCookieController controller = new FortuneCookieController();
  9. To connect the controller to the button, add an action listener to the button like this: 
        button.addActionListener(controller);
  10. Run the program again. Click on the button. What happens? You should get the Woohoo pop-up.

    11. Now make the pop-up deliver a random message


  11. In the actionPerformed method of the controller, generate a random number between 0 and 4: 
       int rand = new Random().nextInt(5);
  12. Depending on the value of rand, show one of the five different fortunes.
  13. [optional] Have other student run your program to receive their fortune prediction.
