Jack-in-the-Box


Goal: Make a program from scratch that has a View (JFrame) and Controller (ActionListener).

The View will have a window (frame) with a single button. A surprise will appear after the button has been clicked 5 times.


    If you need help with any of these steps, take a look at the previous recipes in this module.

    Make the View (Frame with a Button)


  1. In the jack_in_the_box package, create a JackInTheBox class to build a frame wuth a button labeled "Surprise!".
  2. Now make a runner class to run the code above. Run the program. Make sure the frame and button show up before proceeding.

    3. Connect a Controller


  3. Make a Controller class that implements ActionListener.
  4. Add the Controller (ActionListener) to the Surprise button.
  5. Put code in the Controller class, so that when the user presses the surprise button 5 times, it will show them a jack-in-the-box.

    Call the methods below to make a jack-in-the-box picture appear.
    (A sample picture ( jackInTheBox.png ) is provided in the recipe package, but you can use another if preferred)
    private void showPicture(String fileName) {
    try {
        JLabel imageLabel = createLabelImage(fileName);
        JFrame frame = new JFrame();
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.add(imageLabel);
        frame.setVisible(true);
    } catch (Exception e) {
        e.printStackTrace();
    }
}
private JLabel createLabelImage(String fileName) {
	try {
            URL imageURL = getClass().getResource(fileName);
            if (imageURL == null) {
	       System.err.println("Could not find image " + fileName);
	       return new JLabel();
            } else {
                Icon icon = new ImageIcon(imageURL);
                JLabel imageLabel = new JLabel(icon);
                return imageLabel;
            }
        } catch (Exception e) {
            System.err.println("Could not find image " + fileName);
            return new JLabel();
        }
}

  6. Now add code to play a sound when the jack-in-the-box pops up using this method.
    (A sample sound ( homer-woohoo.wav ) is provided in the recipe package but you can use another if preferred) 
    private void playSound(String soundFile) { 
    try {
        AudioClip sound = JApplet.newAudioClip(getClass().getResource(soundFile));
        sound.play();
    } catch (Exception e) {
        e.printStackTrace();
    }
