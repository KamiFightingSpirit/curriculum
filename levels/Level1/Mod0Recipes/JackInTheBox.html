Jack-in-the-Box


Goal: Make a program from scratch that has a View (Swing GUI) and Controller (ActionListener).

The View will have a window (frame) with a single button. A surprise will appear after the button has been clicked 5 times.


    The View


  1. Create a View class to build a frame wuth a button labeled "Surprise!".

    If you need help, take a look at the previous recipes in this module.
  2. Now make a runner class to run the code above. Run the program. Make sure the frame and button show up before proceeding.

    3. Connect a Controller


  3. Make a Controller class that implements ActionListener.

    If you need help, take a look at the previous recipes in this module.
  4. Add the controller to the Surprise button.
  5. Put code in the controller, so that when the user presses the surprise button 5 times, show them a jack-in-the-box.

    Use the methods below to make the jack-in-the-box appear.
    (A sample picture (jackInTheBox.png) is provided in the recipe package, but you can use another)
    private void showPicture(String fileName) {
    try {
        JLabel imageLabel = createLabelImage(fileName);
        JFrame frame = new JFrame();
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.add(imageLabel);
        frame.setVisible(true);
    } catch (Exception e) {
        e.printStackTrace();
    }
}
private JLabel createLabelImage(String fileName) {
	try {
            URL imageURL = getClass().getResource(fileName);
            if (imageURL == null) {
	       System.err.println("Could not find image " + fileName);
	       return new JLabel();
            } else {
                Icon icon = new ImageIcon(imageURL);
                JLabel imageLabel = new JLabel(icon);
                return imageLabel;
            }
        } catch (Exception e) {
            System.err.println("Could not find image " + fileName);
            return new JLabel();
        }
}

    private void playSound(String soundFile) {
